KEY LIST
  - -h or --help -- possible keys  
  - -v or --version -- version of the program
  - -l or --list - list of arguments
  - -n or --name -- file name

Program options: 

  -h or --help                          - messages with program options

  -v or --version                       - version

  -n or --name                          - file name

  -l[<val1>,...] or --list=[<num1>,...] - list of arguments
